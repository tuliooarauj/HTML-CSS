Principais formatações

Negrito / Destaque

Palavra em negrito: NEGRITO usando a tag <b> (NÃO SEMÂNTICA).

Termo em destaque: DESTAQUE usando a tag <STRONG> (SEMÂNTICA).

Itálico / Ênfase

Termo em itálico usando a tag <I> (NÃO SEMÂNTICA).

Termo em ênfase usando a tag <EM> (SEMÂNTICO).

Texto marcado

Pode-se criar um texto marcado com a tag mark.

Texto grande e pequeno

Texto grande e pequeno no parágrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele dever ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Podemos marcar um texto como sublinhado para dar uma ênfase e indicar que ele foi adicionado depois com a tag u (Não semântico).

Para inserir coisas do tipo x2 + 3
